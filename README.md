# 🗨️ Branch Chat App

A real-time group chat application where users can chat based on their university branch (e.g., CSE, ECE, ME). 
Built using **React**, **Node.js**, **Express**, and **Socket.IO**.

---

## 🚀 Features

- 🔐 Authentication check via email (login or register)
- 🧠 Branch-specific chat rooms
- 💬 Real-time messaging using Socket.IO
- 🕓 Chat history fetched from backend
- 🧾 Timestamped messages with user identification
- 🚪 Logout and session control
- 💅 Clean and responsive UI

---

## 🛠️ Tech Stack

### Frontend:
- React
- React Router
- Socket.IO Client

### Backend:
- Node.js
- Express.js
- Socket.IO
- MongoDB (for storing messages and users)
